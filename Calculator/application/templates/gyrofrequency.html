Gyrofrequency

Usage:

gyrofrequency(B, particle[, signed, Z, to_hz]), where
B: The magnetic field magnitude in units convertible to tesla
particle: ‘p’ for protons, ‘D+’ for deuterium, or ‘He-4 +1’ for singly ionized helium-4, etc.
signed: optional, negative for electron, positive for ion. Unsigned(false) means positive
Z: optional, The average ionization (arithmetic mean) for a plasma where the a macroscopic description is valid
to_hz: Set True to to convert function output from angular frequency to Hz

returns omega_c – The particle gyrofrequency in units of radians per second
